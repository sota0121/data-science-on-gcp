## How to learn

- 実装すべきコード（たとえば最終的に利用が推奨されている形のコードなど）を選定して、それだけ実装する
- 各ディレクトリ（章に対応）ごとに用意した `NOTE.md` に学びを記録する
- 開発仮想環境は、各ディレクトリ内で適宜用意する（本家もその粒度で `requirements.txt` を用意している）
